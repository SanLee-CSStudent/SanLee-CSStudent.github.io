Tick Tock

Description: Tick Tock is the name of a creature that lives on your desktop. She will constantly move around the screen to keep you focused. Simple user interface is provided to interact with this creature such as feeding, sleeping, and stroking. Make sure you keep the creature alive; otherwise, it will faint(you can revive it any time).

Sprites for the creature and mouse cursors are from Maplestory: a MMORPG(massively multiplayer online role-playing game).
Use Programming Language: Java.

Eight Puzzle Solver

Description: This program solves for a random eight puzzle. The user can choose which search algorithms to solve a default/user-customized puzzle. Provided search algorithms include Uniform Cost Search, A* Search with Misplaced Tile Heuristics, and A* Search with Euclidean Distance Heuristics.
Warning!!! Finding a solution from an impossible puzzle takes long time.

Use Programming Language: Java.
Link to Project is here.

K-Nearest Neighbor Classifier

Description: This program finds a subset of features with the highest accuracy. While feature search, each subset of features is evaluated through the method of leave-one-out validation. Validating process requires testing an instance with a nearest neighbor classifier. The nearest neighbor classifier requires storing of instances from given datasets, which are under Resources directory, computing distances of an input instance to the rest of instances, and choosing k-number of neighbors to vote on the predicted label of the input. Then, the classifier returns predicted label and compares the result to the real label to measure accuracy.
This program offers two types of search algorithms: forward selection and backward elimination, that are based on greedy algorithm.

Use Programming Language: Java.
